Code for Catって？

コードフォーキャットは、地域猫を地域の資産であると捉え、殺処分（収容数）ゼロを目指している団体の活動を支援するために、IT技術を活用した、持続可能な仕組みを開発し、継続的な支援を行っていきます。

CatBot project

猫に関するあらゆる質問疑問に答えるChatBot（自動応答）サービスです。

猫と人間が長く付き合っていくため、猫に対する正しい知識と正しいつきあい方（飼い方）が分かるようになります。
